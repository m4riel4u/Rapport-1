# Glossaire

Ce glossaire définit les termes importants utilisés dans le projet, en particulier ceux qui pourraient prêter à confusion.

| Terme | Définition |
|-------|------------|
| Avis | Opinion, jugement ou recommendation sur les cours partagé sur la plateforme Discord. |
| Bot Discord | Programme informatique automatisé exécutant des tâches répétitives ou spécifiques sur un serveur Discord. Il catalogue les avis des étudiants sur des cours particuliers grâce à des mots-clés spécifiques. |
| Calendrier académique | Planification des cours, périodes d'inscription, examens, remises de travaux, et congés. |
| Charge de travail | Volume de tâches à accomplir dans un temps donnée pour un cours. |
| Cheminement académique | Parcours suivi par un étudiant dans son programme, comprenant les cours obligatoires et optionnels. |
| Crédit universitaire | Unité de mesure représantant la charge de travail d'un cours universitaire |
| Curriculum | Plan qui définit le contenu, les objectifs, les méthodes d'enseignement et d'évaluation d'un programme. |
| Éligible | Étudiant remplissant les conditions/ critères nécessaires pour avoir accès à un cours comme les cours prérequis, être dans le bon cycle et respecter les contraintes de programme ou statut. |
| Équivalence | Reconnaissance de programmes ou cours universitaires acquis antérieurement. |
| Étudiant | Personne inscrite dans un programme universitaire et comptant s'inscrire à des cours pour la prochainne session. |
| Médianne | Valeur centrale divisant la série des notes académiques ordonnée pour un cours en deux moitiés égales. |
| Moyenne | Résultat de la somme de toutes les notes obtenues dans un cours, divisée par le nombre total de notes. Montre la performance globale des étudiants dans un cours spécifique. |
| Prérequis | Conditions ou compétences nécessaires afin d'avoir accès à quelque chose |
| Résultats agrégés | Donnée résultant d'une synthèse de plusieurs sources. Synthèse des notes des étudiant(e)s pour un cours donné.|
| Session académique | Période durant laquelle se déroule les cours académiques. |
| Seuil minimal | Plus basse valeur requise pour atteindre un objectif ou respecter une condition. Exemple : Il faut 5 avis minimalement pour que les avis d'un cours soient affichés. |
| Sigle de cours | Code court représentant un cours spécifique au sein d'un programme d'études. | 
| TGDE | Technien(ne) en gestion des dossiers étudiants est un personne ressource pour la communauté étudiante assurant le bon déroulement des études et fournissant de l'aide aux étudiants. |
